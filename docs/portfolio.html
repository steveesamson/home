Portfolios

The following is a list of some projects (mostly personal, I have no permissions to share corporate ones.) I had worked on or implemented. Check my repository for an updated list.

  1. TotStream: TotStream is the place for everyone with an occupation to mentor, inspire, do business, learn, acquire new skills, arrange a meet-up, team up, or just hang out to make an imprint in the digital space. Technologies: Node.js, Svelte, Sapper, Redis, Socket.io, Express.js, REST, API, Worker, Event Bus, TypeScript, JavaScript, CSS, HTML.
  2. App Studio: App studio is a platform for visually creating business process flows. Technologies: Node.js, Svelte, JavaScript, CSS, HTML.
  3. CronTimex: CronTimex is a dependencies-free Cron Time Expression Generator/Builder written in Typescript. Technologies: TypeScript.
  4. Microform: Microform is a tiny library for managing forms in svelte/sveltekit. Technologies: Node.js, Svelte, SvelteKit, TypeScript, JavaScript, CSS, HTML.
  5. Svesta: Svesta is a tiny svelte state management library for svelte and sveltekit. Technologies: Node.js, Svelte, SvelteKit, TypeScript, JavaScript, CSS, HTML.
  6. Slicks: Slicks is a simple javascript Client MVC framework for Single-Page-Applications, SPAs. It uses seleto.js, stud.js for rendering pre-compiled templates. Technologies: Node.js, JavaScript, CSS, HTML.
  7. Slick-mysql: Slicks-mysql allows the expressive writing of database queries and routines. slicks-mysql permit chaining, which is intuitive as you can nearly guess what should come next even if you are just getting started with slicks-mysql. slicks-mysql is not an ORM. It was developed to allow folks coming from relational databases background write expressive queries with object interactions in mind. Inspired by Codeigniter Active Record. Technologies: Node.js, MySQL, Database, JavaScript.
  8. slicks-postgres: slicks-postgres allows the expressive writing of database queries and routines for PostgreSQL. slicks-postgres permit chaining, which is intuitive as you can nearly guess what should come next even if you are just getting started with slicks-postgres. slicks-postgres is not an ORM. It was developed to allow folks coming from relational databases background write expressive queries with object interactions in mind. Inspired by Codeigniter Active Record. Technologies: Node.js, PostgreSQL, Database, JavaScript.
  9. Secret Escapes Search: This is a fullstack challenge implementation for `Secret Escape Portal`. Technologies: Docker, Node.js, GraphQL, React.js, TypeScript.
  10. REDSpace swapi: This is a fullstack challenge implementation for `REDSpace Swapi`. Technologies: Docker, Node.js, Express.js, REST API, React.js, TypeScript.
  11. Fitbitz: A simple app showcasing Apollo graphQL Server and Client, ApolloServer, Express and Knex with PostgreSQL. Technologies: Node.js, GraphQL, React.js, Express.js, ApolloServer, TypeScript, PostgreSQL.
  12. Stud: A very simple and fast template engine for compiling and rendering pre-compiled HTML templates in an unopinionated fashion.Technologies: JavaScript.
